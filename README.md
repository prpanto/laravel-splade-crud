## Build with:
1. Laravel
2. Splade
3. Tailwind

## Installation:

1. `git clone https://github.com/propanto/laravel-splade-crud`
2. `composer install`
3. `cp .env.example .env`
4. Set database configs on `.env` file
5. `php artisan key:generate`
6. `php artisan migrate --seed`
7. `npm i & npm run dev`
